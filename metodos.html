Date.prototype[@@toPrimitive]()

O método Date.prototype[@@toPrimitive]() é uma função especial associada ao símbolo Symbol.toPrimitive que é chamada internamente pelos algoritmos do JavaScript durante operações que exigem a conversão de um objeto Date para um valor primitivo. Isso geralmente ocorre quando um objeto Date é utilizado em contextos que esperam um valor primitivo, como operações de comparação ou coerção.

Este método oferece a oportunidade de personalizar como um objeto Date é convertido para um valor primitivo, permitindo um controle mais preciso sobre seu comportamento em diferentes situações.

Assinatura

Date.prototype[@@toPrimitive](hint: string): number | string | default;

O método leva um argumento hint, que é uma string indicando o tipo de conversão desejada. Os valores possíveis para hint são:

  • 'number': Solicita a conversão para um número.
  • 'string': Solicita a conversão para uma string.
  • 'default': Utilizado em operações que não especificam um hint particular.

Durante a implementação do método @@toPrimitive, você pode personalizar o comportamento do objeto Date para diferentes cenários de conversão. Aqui está um exemplo que demonstra como você pode implementar o método e utilizar diferentes hints: 


// Implementação do método @@toPrimitive
Date.prototype[Symbol.toPrimitive] = function (hint) {
    if (hint === 'number') {
        return this.getTime();
    } else if (hint === 'string') {
        return this.toISOString();
    } else {
        // Lida com o hint 'default'
        return this.toString();
    }
};

// Exemplo de uso
const currentDate = new Date();
resultado += +currentDate + '\n'; // Converte para número usando o hint 'number'
resultado += String(currentDate) + '\n'; // Converte para string usando o hint 'string'
resultado += currentDate + '\n'; // Converte usando o hint 'default'

Date.prototype.getDate()

O método Date.prototype.getDate() retorna o dia do mês (de 1 a 31) para a data representada pelo objeto Date.

Assinatura

Date.prototype.getDate(): number;

Este método não requer argumentos. Ele retorna um número representando o dia do mês.

Quando utilizado, o método getDate() considera o fuso horário local.

Exemplos de Uso

Date.prototype.getDay()

O método Date.prototype.getDay() retorna o dia da semana (de 0 a 6) para a data representada pelo objeto Date. Os dias da semana são representados pelos números de 0 a 6, onde 0 corresponde a domingo e 6 corresponde a sábado.

Assinatura

Date.prototype.getDay(): number;

Este método não requer argumentos. Ele retorna um número representando o dia da semana.

Quando utilizado, o método getDay() considera o fuso horário local.

Exemplos de Uso

Date.prototype.getFullYear()

O método Date.prototype.getFullYear() retorna o ano de quatro dígitos para o objeto Date. O ano retornado é o ano completo, não ajustado para um século específico (por exemplo, 19 para 2019).

Assinatura

Este método não leva nenhum argumento. Simplesmente retorna o ano de quatro dígitos como um número.
