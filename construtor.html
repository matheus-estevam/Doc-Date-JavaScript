Construtor Date()

Objetivo: O construtor Date() é utilizado para criar instâncias do objeto Date em JavaScript, permitindo a manipulação de datas e horas.

Sintaxe:

        
          new Date();                        // Data e hora atuais
          new Date(valor);                   // Milissegundos desde 1º de janeiro de 1970 (Epoch)
          new Date(dataString);              // String representando uma data
          new Date(ano, mês, dia, hora, minuto, segundo, milissegundo);  // Data e hora específicas
